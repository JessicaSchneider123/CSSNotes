CSS Selectors and Specificity

-Selectors are sometimes called tag name or element selector

Types of Selectors

Syntax Name Description Example
* universal selector selects all elements of any type * {color: red;}
.class Class selects all elements with class name <h1 class="green bold">
note: space between classes
#id ID Can only have a single value and only used once per page
overrides types & classes		 <h1 id="large-title">
[attribute] attribute selector can be used t target HTML elements that already contain attributes.
allows elements of the same type to be targeted differently by attribute or attribute value		 img[src*="winter"] selects all images with "winter" in the source name
:focus
:visited
:disabled
:active		 pseudo-class changes appearance of elements based on user interactions a:hover{orange} turns all anchor elements orange on hover
-note no space between "a" and ""hover"

Chaining

Syntax Description
<div class= value1 value2 value3> </div> some HTML attributes can have multple values, separated by a space
h1.special
(no space)		 Selects only <h1> elements with a class of special. If a <p> also had a class of special, this rule would not style it
h1, h2
(comma, space)		 selects all <h1> and <h2> elements
.main-list li
(space, no comma)		 selects all <li> elements under main-list class
